# janstova.cz

Osobní portfolio Kláry Janstové: DevOps, cloud infrastruktura, CI/CD, Kubernetes a Java/Spring Boot backend.

## Lokální spuštění

Web je statický. Stačí otevřít `index.html` v prohlížeči, případně použít libovolný jednoduchý static server.

## Nasazení na Netlify

1. Vytvořit GitHub repozitář a nahrát tento projekt.
2. V Netlify zvolit `Add new site` -> `Import an existing project`.
3. Vybrat GitHub repozitář.
4. Build command nechat prázdný.
5. Publish directory nastavit na kořen repozitáře, tedy `.`.
6. Po deploy přidat vlastní doménu `janstova.cz`.
